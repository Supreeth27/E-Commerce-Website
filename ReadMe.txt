This is a E-Commerce website for the Sports Equipment. A user can login into the website and can add the item into his userprofile. 
User can further perform update and delete operations in user profile.

Technical Stack:
Node.js for backend,
MongoDB as database
EJS for frontend

How to run:
For running this application navigate to folder containing this project and run the following commands 
i) npm install(For installing dependencies)
ii) node app.js or nodemon app.js
Make sure to install MongoDB and run MongoDB in terminal. 
Then, open a new window in terminal and type for running Mongo shell.
Then copy and paste the commands in hw5_create_db.txt for creating a database in MongoDB.
